What this document validates

We are updating a mate-choice-copying meta-analysis built on two existing datasets that overlap in the primary studies they cover but report different effect-size metrics:

  • Davies et al. (2020) — Hedges’ d
  • Jones & DuVal (2019) — log odds ratio (lnOR)

Before pooling them we need to know: for the studies both teams extracted, do the effect sizes agree once placed on a common scale? This is a method-comparison problem (two “instruments” measuring the same underlying effects), so — unlike a single-team re-extraction audit — we can apply formal agreement statistics (correlation, Bland–Altman limits of agreement) on top of the per-study qualitative review.

The matching itself (study-level crosswalk + within-study optimal effect-size assignment) is performed by crosscheck_davies_jones.py; this report reads its output and assesses agreement. Key methodological points carried over:

  • Scale conversion. Jones lnOR is mapped to Hedges’ d via the logistic-to-normal approximation \(d = \text{lnOR}\times\sqrt{3}/\pi\) (≈ 0.5513; Borenstein et al. 2009).
  • Absolute magnitudes. The two datasets use opposite sign conventions (Davies’ non-directional d carries the raw mean-difference sign; Jones’ lnOR sign encodes copying direction), so agreement is assessed on \(|d|\). Sign concordance is reported separately as a diagnostic.

Re-extraction: what happens when we return to the source PDFs

The agreement analysis flags where the two datasets disagree, but not which is correct. To resolve that for the worst cases, we pulled the primary-study PDFs for every study whose effect-size pair disagreed by > 0.5 Hedges-d units and re-extracted each effect size from the raw data, recomputing it as a 2×2 odds ratio under the meta-analysis’s own conventions (treatment-vs-control where the study used a control group; proportion-vs-chance otherwise). This gives a third, independent estimate per effect size — a tie-breaker.

Show R code 
reext <- read_csv(file.path(out_dir, "reextraction_flagged_7studies.csv"),

    mutate(
        study     = gsub("_", " ", sub("^([^_]+_[0-9]+).*", "\\1", identifierStudyId)),
        label     = paste0(study, " · ", identifierExperimentId),
        reext_d   = log(effectSizeCalculatedEstimate) * 0.5513,   # ln(OR) -> Hedges d

        davies_d  = compare_davies_hedges_d,
        reext_abs = abs(reext_d), jones_abs = abs(jones_d), davies_abs = abs(davies_d)

Across the 12 re-extracted effect sizes the point estimates sit almost exactly on the Jones values (mean |Δd| = 0.00) and far from the Davies values (mean |Δd| = 0.83). Two cautions stop this from being read as “the extraction is validated against Jones”:

  • Point-estimate agreement is expected, not corroborating. These outcomes are discrete choice counts read directly from each PDF; any careful extractor building the same 2×2 obtains the same lnOR. Agreement here confirms we read the papers the same way — it is not independent evidence that either prior extraction is correct.
  • The original crosswalk selected the construction that matched Jones. For each study it chose the 2×2 that reproduced Jones’s variance, which makes the variance match circular. The next section removes that circularity: it re-derives every effect size from a rule fixed by the study design before any comparison to Jones, and reports count agreement and design agreement separately.

Count agreement vs design agreement (and a correction to Jones)

To break the circularity we re-derive each effect size from a rule fixed by the study design alone — before looking at Jones — and report two things separately: count agreement (does our lnOR from the raw counts match Jones, i.e. did we read the paper the same way) and design agreement (does our variance and N match, i.e. did we model the design the same way). The rule has two branches:

  • Real control group — the 2×2 has distinct treatment and control cells (e.g. Dugatkin 1992’s naïve non-observer group). A standard 2×2 odds ratio is valid: \(\mathrm{Var}=1/A+1/B+1/C+1/D\), \(N=A+B+C+D\).
  • Single group vs 50 % chance — the “50/50 rule” sets \(C=D=n/2\). The design-faithful estimator is the one-sample logit of the observed proportion: \(\mathrm{Var}=1/A+1/B\), \(N=A+B\). The 50/50 group is a phantom control: it invents \(n\) observations that were never collected and inflates the variance by exactly \(4/n\) while doubling the reported \(N\).
Show R code 
reext2 <- reext %>%
    mutate(
        A = effectSizeSampleSizeA, B = effectSizeSampleSizeB,
        C = effectSizeSampleSizeC, D = effectSizeSampleSizeD,
        design = if_else(C == D, "single group vs 50% chance", "real control group"),
        corr_var = if_else(design == "real control group",
                           1/A + 1/B + 1/C + 1/D, 1/A + 1/B),
        corr_N   = if_else(design == "real control group", A + B + C + D, A + B),
        jones_var    = compare_jones_VlnOR,
        count_agree  = abs(reext_d - jones_d) < 0.02,   # point estimate (lnOR × √3/π)
        design_agree = abs(corr_var - jones_var) < 0.01
    )

reext2 %>% summarise(

The corrected \(N\) also reframes Davies’s \(N\) errors below: Davies’s \(N=40\) for Dugatkin 1993 is wrong against the design-faithful \(N=20\), not merely against Jones.

Why Davies diverges — four failure modes

The re-extraction shows the discrepancies are not random noise or conversion error; they trace to four identifiable problems, all on the Davies side:

  1. Different outcome construct. For Applebaum & Cruz 2000 and Briggs et al. 1996, Davies extracted a secondary affiliation-time measure (often non-significant) rather than the preference-reversal frequency the studies analyse; for Howard et al. 1998 Davies extracted a male-size preference — a different question entirely.
  2. Computational artifacts. Dugatkin 1992’s Davies d = 3.732 is exactly 2 + √3, the signature of a degenerate-2×2 escalc input; Gierszewski et al. 2018’s d = 2.43 is inflated by the structural zero cell at first choice.
  3. Sign error. Howard et al. 1998: Davies records +1.14 for a study that found no copying (females retained their original preference 15:5); the re-extraction and Jones both give −0.61.
  4. Pool-vs-split, not an error. Fowler-Finn et al. 2015 is the one case where neither is wrong: Davies pooled one effect (d 0.63) where Jones — and our re-extraction — split it into a copying arm (d 1.21) and a no-copying arm (d 0.10), matching the study’s significant phenotype × age interaction.

A design subtlety surfaced during re-extraction: for Dugatkin 1992 the copying tests are correctly compared against the study’s naïve control group, not against chance. Under that (correct) design Davies’s total N = 40 is right — only the magnitude (the 2 + √3 artifact) is wrong — while Jones’s third recorded value for this study (lnOR 2.234) is spurious: it reproduces from no pair of the paper’s six experiments and should be dropped.

What to report in the manuscript

  • Re-extraction (the > 0.5 set). Returning to the source PDFs for the 12 effect sizes that disagreed most, the design-blind re-derivation reproduced the point estimates wherever the data are discrete counts (mean |Δd| vs Jones = 0.00; vs Davies = 0.83), corrected Jones on the single-group / vs-chance studies (a 4/n phantom-control variance inflation from the 50/50 rule, with halved N), and isolated Davies as the divergent extraction — via different outcome constructs, escalc artifacts, a sign error, and one defensible pool-vs-split. This is an independent re-derivation, not a validation against Jones: the point-estimate agreement is expected for discrete counts and the variance was no longer reverse-engineered to match.
  • Overlap. Of 37 Jones & DuVal studies, 29 are shared with Davies; 8 are unique to Jones (28 effect sizes).
  • Agreement. Across 52 comparable effect-size pairs, Jones-converted and Davies magnitudes correlate at Pearson r = 0.79 (65% within tolerance), rising to r = 0.89 among studies where both teams extracted the same number of effect sizes.
  • Direction of disagreement. Mean Bland–Altman bias = -0.14 d-units, indicating Davies reports slightly larger magnitudes on average; most large discrepancies trace to different outcomes extracted (behaviour-type columns) or different effect-size partitioning, not to the metric conversion.

What this document validates

We are updating a mate-choice-copying meta-analysis built on two existing datasets that overlap in the primary studies they cover but report different effect-size metrics:

  • Davies et al. (2020) — Hedges’ d
  • Jones & DuVal (2019) — log odds ratio (lnOR)

Before pooling them we need to know: for the studies both teams extracted, do the effect sizes agree once placed on a common scale? This is a method-comparison problem (two “instruments” measuring the same underlying effects), so — unlike a single-team re-extraction audit — we can apply formal agreement statistics (correlation, Bland–Altman limits of agreement) on top of the per-study qualitative review.

The matching itself (study-level crosswalk + within-study optimal effect-size assignment) is performed by crosscheck_davies_jones.py; this report reads its output and assesses agreement. Key methodological points carried over:

  • Scale conversion. Jones lnOR is mapped to Hedges’ d via the logistic-to-normal approximation \(d = \text{lnOR}\times\sqrt{3}/\pi\) (≈ 0.5513; Borenstein et al. 2009).
  • Absolute magnitudes. The two datasets use opposite sign conventions (Davies’ non-directional d carries the raw mean-difference sign; Jones’ lnOR sign encodes copying direction), so agreement is assessed on \(|d|\). Sign concordance is reported separately as a diagnostic.

Re-extraction: what happens when we return to the source PDFs

The agreement analysis flags where the two datasets disagree, but not which is correct. To resolve that for the worst cases, we pulled the primary-study PDFs for every study whose effect-size pair disagreed by > 0.5 Hedges-d units and re-extracted each effect size from the raw data, recomputing it as a 2×2 odds ratio under the meta-analysis’s own conventions (treatment-vs-control where the study used a control group; proportion-vs-chance otherwise). This gives a third, independent estimate per effect size — a tie-breaker.

Across the r nrow(reext) re-extracted effect sizes the point estimates sit almost exactly on the Jones values (mean |Δd| = r fmt(mj,2)) and far from the Davies values (mean |Δd| = r fmt(md,2)). Two cautions stop this from being read as “the extraction is validated against Jones”:

  • Point-estimate agreement is expected, not corroborating. These outcomes are discrete choice counts read directly from each PDF; any careful extractor building the same 2×2 obtains the same lnOR. Agreement here confirms we read the papers the same way — it is not independent evidence that either prior extraction is correct.
  • The original crosswalk selected the construction that matched Jones. For each study it chose the 2×2 that reproduced Jones’s variance, which makes the variance match circular. The next section removes that circularity: it re-derives every effect size from a rule fixed by the study design before any comparison to Jones, and reports count agreement and design agreement separately.

Count agreement vs design agreement (and a correction to Jones)

To break the circularity we re-derive each effect size from a rule fixed by the study design alone — before looking at Jones — and report two things separately: count agreement (does our lnOR from the raw counts match Jones, i.e. did we read the paper the same way) and design agreement (does our variance and N match, i.e. did we model the design the same way). The rule has two branches:

  • Real control group — the 2×2 has distinct treatment and control cells (e.g. Dugatkin 1992’s naïve non-observer group). A standard 2×2 odds ratio is valid: \(\mathrm{Var}=1/A+1/B+1/C+1/D\), \(N=A+B+C+D\).
  • Single group vs 50 % chance — the “50/50 rule” sets \(C=D=n/2\). The design-faithful estimator is the one-sample logit of the observed proportion: \(\mathrm{Var}=1/A+1/B\), \(N=A+B\). The 50/50 group is a phantom control: it invents \(n\) observations that were never collected and inflates the variance by exactly \(4/n\) while doubling the reported \(N\).

#| label: design-rederivation
reext2 <- reext %>%
    mutate(
        A = effectSizeSampleSizeA, B = effectSizeSampleSizeB,
        C = effectSizeSampleSizeC, D = effectSizeSampleSizeD,
        design = if_else(C == D, "single group vs 50% chance", "real control group"),
        corr_var = if_else(design == "real control group",
                           1/A + 1/B + 1/C + 1/D, 1/A + 1/B),
        corr_N   = if_else(design == "real control group", A + B + C + D, A + B),
        jones_var    = compare_jones_VlnOR,
        count_agree  = abs(reext_d - jones_d) < 0.02,   # point estimate (lnOR × √3/π)
        design_agree = abs(corr_var - jones_var) < 0.01
    )

reext2 %>% summarise(
    `Effect sizes` = n(),
    `Count agreement (lnOR matches Jones)`      = sum(count_agree),

) %>% pivot_longer(everything(), names_to = "Quantity", values_to = "n") %>%

For every single-group study except Howard 1998, our Var + 4/n reproduces Jones’s variance to the fourth decimal, confirming the inflation is the invented control group rather than a different reading. Howard 1998 is the exception (Jones used a different null), and our estimate is preferred there. So on the vs-chance studies the design-faithful re-derivation corrects Jones — smaller variance, halved \(N\) — rather than reproducing him: the earlier “matches Jones exactly” was, for these studies, matching a 50/50-rule artifact. The corrected \(N\) also reframes Davies’s \(N\) errors below: Davies’s \(N=40\) for Dugatkin 1993 is wrong against the design-faithful \(N=20\), not merely against Jones.

Why Davies diverges — four failure modes

The re-extraction shows the discrepancies are not random noise or conversion error; they trace to four identifiable problems, all on the Davies side:

  1. Different outcome construct. For Applebaum & Cruz 2000 and Briggs et al. 1996, Davies extracted a secondary affiliation-time measure (often non-significant) rather than the preference-reversal frequency the studies analyse; for Howard et al. 1998 Davies extracted a male-size preference — a different question entirely.
  2. Computational artifacts. Dugatkin 1992’s Davies d = 3.732 is exactly 2 + √3, the signature of a degenerate-2×2 escalc input; Gierszewski et al. 2018’s d = 2.43 is inflated by the structural zero cell at first choice.
  3. Sign error. Howard et al. 1998: Davies records +1.14 for a study that found no copying (females retained their original preference 15:5); the re-extraction and Jones both give −0.61.
  4. Pool-vs-split, not an error. Fowler-Finn et al. 2015 is the one case where neither is wrong: Davies pooled one effect (d 0.63) where Jones — and our re-extraction — split it into a copying arm (d 1.21) and a no-copying arm (d 0.10), matching the study’s significant phenotype × age interaction.

A design subtlety surfaced during re-extraction: for Dugatkin 1992 the copying tests are correctly compared against the study’s naïve control group, not against chance. Under that (correct) design Davies’s total N = 40 is right — only the magnitude (the 2 + √3 artifact) is wrong — while Jones’s third recorded value for this study (lnOR 2.234) is spurious: it reproduces from no pair of the paper’s six experiments and should be dropped.

What to report in the manuscript

  • Re-extraction (the > 0.5 set). Returning to the source PDFs for the r nrow(reext) effect sizes that disagreed most, the design-blind re-derivation reproduced the point estimates wherever the data are discrete counts (mean |Δd| vs Jones = r fmt(mj,2); vs Davies = r fmt(md,2)), corrected Jones on the single-group / vs-chance studies (a 4/n phantom-control variance inflation from the 50/50 rule, with halved N), and isolated Davies as the divergent extraction — via different outcome constructs, escalc artifacts, a sign error, and one defensible pool-vs-split. This is an independent re-derivation, not a validation against Jones: the point-estimate agreement is expected for discrete counts and the variance was no longer reverse-engineered to match.
  • Overlap. Of r nrow(shared) + nrow(unmatched) Jones & DuVal studies, r n_shared are shared with Davies; r nrow(unmatched) are unique to Jones (r sum(unmatched$n_effect_sizes) effect sizes).
  • Agreement. Across r n_pairs comparable effect-size pairs, Jones-converted and Davies magnitudes correlate at Pearson r = r fmt(pear$estimate,2) (r fmt(100*n_match/n_pairs,0)% within tolerance), rising to r = r fmt(pear_sub$estimate,2) among studies where both teams extracted the same number of effect sizes.
  • Direction of disagreement. Mean Bland–Altman bias = r fmt(bias,2) d-units, indicating Davies reports r if (bias < 0) "slightly larger" else "slightly smaller" magnitudes on average; most large discrepancies trace to different outcomes extracted (behaviour-type columns) or different effect-size partitioning, not to the metric conversion.
